## 1. 产品概述

### 1.1 定位
AI便利贴是一款便携式智能显示设备，通过磁吸或支架方式可灵活放置于冰箱、桌面等场景，提供生活信息展示和语音交互能力。

### 1.2 核心价值
- **低侵入感**：实体设备，无需打开手机
- **无感自然**：墨水屏常显，静默更新
- **语音驱动**：长按即说，简化操作
- **长期续航**：2-4周一充，真正的无感体验

## 3. 功能需求

### 3.1 页面系统（8个页面）

| 页面 | 显示内容 | 模板支持 | 语音控制 |
|------|----------|----------|----------|
| 日期 | 大号日期、星期、农历 | 标准/极简/节日 | 农历开关、字体大小 |
| 天气 | 图标、当前温度、高低温 | 标准/详情/极简 | 显示项目切换、单位切换 |
| 待办 | 清单列表（最多5-7项） | 列表/筛选视图 | 增删改查、排序、归档 |
| 日历 | 月历/周历/日视图 | 月/周/日/议程 | 视图切换、农历/假期开关 |
| 倒计时 | 事件名称+大数字 | 多种显示样式 | 新建/删除/调整目标日期 |
| 节气 | 当前节气+意象图标 | 单模板 | - |
| 电量 | 百分比+预估续航天数 | 单模板 | - |
| 同步 | 最后同步时间+状态 | 单模板 | 手动同步、重置连接 |

### 3.2 交互逻辑

**按钮操作**：
- **单击**：正向循环切换页面
- **双击**：反向循环切换页面  
- **长按**（>800ms）：进入语音输入模式

**提示音**：
- 单击/双击：短促"滴"声
- 长按触发：上升音"叮~"
- 识别完成：下降音"咚"
- 错误："嘟嘟"两声

### 3.3 语音能力

**支持场景**：
- 页面导航："切换到天气"、"下一页"
- 模板切换："显示详细天气"、"切换到周视图"
- 数据操作："添加待办买牛奶"、"设置考试倒计时30天"
- 系统控制："立即同步"、"重置网络"

**技术方案**：
- 本地VAD检测（Silero-VAD）
- 云端ASR（豆包/千问流式识别）
- MCP协议调用Skill Tools

## 4. 非功能需求

### 4.1 性能
- 页面切换响应 < 300ms
- 语音处理延迟 < 2秒
- 墨水屏刷新时间 < 1.5秒

### 4.2 续航
- 正常使用：2-4周
- 深度休眠：可达3个月
- 语音交互：每次约消耗0.1%电量

### 4.3 网络
- WiFi连接稳定性 > 99%
- 同步间隔：5-15分钟可配置
- 离线模式支持本地缓存

## 5. 技术架构

### 5.1 硬件架构
```
ESP32-S3
├── 4.2寸墨水屏（400x300）
├── 麦克风（I2S）
├── 扬声器（I2S）
├── 单按钮
└── 锂电池（2000mAh预估）
```

### 5.2 软件架构
```
ESP32端（轻量客户端）
├── 状态机引擎
├── 页面渲染器
├── MCP客户端
└── 本地存储（SPIFFS）

服务器端（Skill Host）
├── MCP Server（FastMCP）
├── 8个预置Skill
├── LLM Router（豆包/千问/Claude）
└── 数据存储（SQLite/Redis）
```

### 5.3 Skill列表
1. **DateSkill** - 日期显示管理
2. **WeatherSkill** - 天气数据获取与展示
3. **TodoSkill** - 待办事项CRUD
4. **CalendarSkill** - 日历视图与事件
5. **CountdownSkill** - 倒计时管理
6. **SolarTermSkill** - 节气计算与显示
7. **BatterySkill** - 电量监控与预估
8. **SyncSkill** - 同步控制与状态

## 7. 风险与缓解

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|----------|
| 墨水屏刷新慢影响体验 | 中 | 高 | 优化局部刷新，减少全屏刷新 |
| 语音识别准确率不足 | 高 | 中 | 支持多模型切换，优化提示词 |
| 续航不达预期 | 高 | 中 | 深度休眠策略，功耗优化 |
| WiFi连接不稳定 | 中 | 中 | 断线重连机制，离线模式 |
