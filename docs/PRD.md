## 1. 产品概述

### 1.1 定位
AI便利贴是一款便携式智能显示设备，通过磁吸或支架方式可灵活放置于冰箱、桌面等场景，提供生活信息展示和语音交互能力。

### 1.2 核心价值
- **低侵入感**：实体设备，无需打开手机
- **无感自然**：墨水屏常显，静默更新
- **语音驱动**：长按即说，简化操作
- **长期续航**：2-4周一充，真正的无感体验

| ID | 故事 | 优先级 |
|----|------|--------|
| US-1 | 作为用户，我想通过语音添加待办事项，以便快速记录想法 | Must |
| US-2 | 作为用户，我想查看考试倒计时，以便保持目标感 | Must |
| US-3 | 作为用户，我想了解今日天气，以便决定穿着 | Must |
| US-4 | 作为用户，我想通过语音切换显示模板，以便个性化展示 | Should |

## 3. 功能需求

### 3.1 页面系统（8个页面）

| 页面 | 显示内容 | 模板支持 | 语音控制 |
|------|----------|----------|----------|
| 日期 | 大号日期、星期、农历 | 标准/极简/节日 | 农历开关、字体大小 |
| 天气 | 图标、当前温度、高低温 | 标准/详情/极简 | 显示项目切换、单位切换 |
| 待办 | 清单列表（最多5-7项） | 列表/筛选视图 | 增删改查、排序、归档 |
| 日历 | 月历/周历/日视图 | 月/周/日/议程 | 视图切换、农历/假期开关 |
| 倒计时 | 事件名称+大数字 | 多种显示样式 | 新建/删除/调整目标日期 |
| 节气 | 当前节气+意象图标 | 单模板 | - |
| 电量 | 百分比+预估续航天数 | 单模板 | - |
| 同步 | 最后同步时间+状态 | 单模板 | 手动同步、重置连接 |

### 3.1.1 页面详细设计

#### 日期页面 (Date)
**标准模板布局** (400x300像素，纵向):
```
┌──────────────────────┐
│     2026年3月       │ ← 12px, 顶部居中
├──────────────────────┤
│                      │
│        13           │ ← 72px, 超大数字居中
│                      │
├──────────────────────┤
│       星期五         │ ← 24px, 星期
│    农历二月初六      │ ← 14px, 农历（可开关）
│                      │
│    ★ 龙抬头 ★      │ ← 12px, 节日/节气提醒
└──────────────────────┘
```

**字段定义**:
| 字段 | 格式 | 来源 | 更新频率 |
|------|------|------|----------|
| 年月 | YYYY年M月 | 本地RTC | 实时 |
| 日 | D | 本地RTC | 实时 |
| 星期 | 星期X | 本地计算 | 实时 |
| 农历 | 农历X月X | 本地算法/服务器 | 日 |
| 节日 | 节日名称 | 本地数据库 | 日 |
| 节气 | 节气名称 | 本地算法 | 节气日 |

**极简模板**: 只显示日期+星期，隐藏农历和节日，字更大
**节日模板**: 节日/节气放大显示，日期缩小

#### 同步页面 (Sync)
**单模板布局**:
```
┌──────────────────────┐
│                      │
│     [同步图标]       │ ← 同步状态图标
│                      │
│      已同步 ✓        │ ← 状态文字
│                      │
│ 上次同步: 14:32     │ ← 时间
│                      │
│ WiFi: 已连接         │ ← 网络状态
│ 本地数据: 23条       │ ← 本地缓存统计
└──────────────────────┘
```

**字段定义**:
| 字段 | 格式 | 来源 | 说明 |
|------|------|------|------|
| 同步状态 | 已同步/同步中/失败 | 系统状态 | 图标+文字 |
| 上次时间 | HH:MM | 本地记录 | 24小时制 |
| WiFi状态 | 已连接/未连接 | 网络检测 | 显示SSID |
| 信号强度 | 图标 | 测量 | 4档信号 |
| 本地数据 | X条 | 统计 | 待办+倒计时总数 |
| 待同步 | X条 | 统计 | 有变更未同步 |
| 设备ID | 后4位 | 配置 | 多设备区分 |

**状态说明**:
- 已同步 ✓: 绿色图标，数据最新
- 同步中 ⟳: 旋转动画，正在同步
- 失败 ✗: 红色图标，可手动重试
- 离线: 灰色图标，无网络连接

### 3.2 交互逻辑

**按钮操作**：
- **单击**：正向循环切换页面
- **双击**：反向循环切换页面  
- **长按**（>800ms）：进入语音输入模式

**提示音**：
- 单击/双击：短促"滴"声
- 长按触发：上升音"叮~"
- 识别完成：下降音"咚"
- 错误："嘟嘟"两声

### 3.3 语音能力

**支持场景**：
- 页面导航："切换到天气"、"下一页"
- 模板切换："显示详细天气"、"切换到周视图"
- 数据操作："添加待办买牛奶"、"设置考试倒计时30天"
- 系统控制："立即同步"、"重置网络"

**技术方案**：
- 本地VAD检测（Silero-VAD）
- 云端ASR（豆包/千问流式识别）
- MCP协议调用Skill Tools

## 4. 非功能需求

### 4.1 性能
- 页面切换响应 < 300ms
- 语音处理延迟 < 2秒
- 墨水屏刷新时间 < 1.5秒

### 4.2 续航
- 正常使用：2-4周
- 深度休眠：可达3个月
- 语音交互：每次约消耗0.1%电量

### 4.3 网络
- WiFi连接稳定性 > 99%
- 同步间隔：5-15分钟可配置
- 离线模式支持本地缓存

## 5. 技术架构

### 5.1 硬件架构
```
ESP32-S3
├── 4.2寸墨水屏（400x300）
├── 麦克风（I2S）
├── 扬声器（I2S）
├── 单按钮
└── 锂电池（2000mAh预估）
```

### 5.2 软件架构
```
ESP32端（轻量客户端）
├── 状态机引擎
├── 页面渲染器（JSON驱动）
├── WebSocket客户端
└── 本地存储（SPIFFS）

服务器端（Pydantic AI + MCP）
├── WebSocket Gateway
├── 豆包ASR（语音识别）
├── Pydantic AI Agent（单模型）
├── MCP Tools（8个页面Tools + PageManager）
└── 数据存储（MongoDB/SQLite）
```

### 5.3 MCP Tools列表
1. **Date Tools** - 日期显示管理
2. **Weather Tools** - 天气数据获取与展示
3. **Todo Tools** - 待办事项CRUD
4. **Calendar Tools** - 日历视图与事件
5. **Countdown Tools** - 倒计时管理
6. **SolarTerm Tools** - 节气计算与显示
7. **Battery Tools** - 电量监控与预估
8. **Sync Tools** - 同步控制与状态
9. **PageManager Tools** - 页面管理（创建/隐藏/列举）
