# UI显示规范

## 1. 屏幕状态指示

#### 状态栏设计（所有页面通用）
```
┌──────────────────────┐
│● WiFi  🔋85%        │ ← 左上角状态栏
│                      │
│     页面内容...      │
│                      │
└──────────────────────┘
```

**状态栏元素**:
| 图标 | 显示条件 | 含义 |
|------|---------|------|
| ● | WiFi连接 | 在线状态 |
| ○ | WiFi断开 | 离线状态 |
| ⟳ | 同步中 | 正在同步 |
| ✓ | 已同步 | 同步完成 |
| ✗ | 同步失败 | 上次同步失败 |
| 🔋XX% | 始终显示 | 电量百分比 |
| 🔌 | 充电中 | 正在充电 |

#### 全屏状态提示
| 状态 | 显示内容 | 持续时间 | 可取消 |
|------|---------|----------|--------|
| 同步中 | "⟳ 同步中..." | 直到完成 | 否 |
| 同步失败 | "✗ 同步失败，单击重试" | 5秒或单击 | 是 |
| 电量低 | "🔋 电量不足，请及时充电" | 5秒 | 是 |
| 网络错误 | "○ 网络异常" | 5秒或单击 | 是 |
| 语音模式 | "🎤 请说话..." | 直到结束 | 是(单击) |

## 4. 异常场景处理

### 4.1 设备启动流程
```
上电 → 硬件自检 → 加载配置 → 连接WiFi → 同步数据 → 进入待机
   │       │          │          │          │
   ▼       ▼          ▼          ▼          ▼
 失败    失败       失败      重试3次    允许离线
  →红灯快闪  →安全模式  →使用默认  →提示离线 →进入待机
```

### 4.2 死机/卡死处理
- **检测**: 看门狗定时器5秒超时
- **恢复**: 自动重启
- **提示**: 重启后显示"系统恢复"

### 4.3 存储满处理
- **检测**: SPIFFS剩余<10%
- **处理**: 自动清理最旧的日志文件
- **提示**: 显示"存储空间不足"

### 4.4 内存不足处理
- **检测**: 剩余堆内存<20KB
- **处理**: 拒绝新的语音操作，强制同步后重启
- **提示**: 显示"内存不足，重启中"

### 4.5 按钮卡住处理
- **检测**: 按钮按下超过30秒
- **处理**: 视为硬件故障，忽略按钮输入
- **恢复**: 松开按钮后恢复正常

## 5. 交互时序参考

### 5.1 页面切换时序
```
0ms      50ms     100ms    300ms    1500ms   1600ms
 │         │        │        │        │        │
 ▼         ▼        ▼        ▼        ▼        ▼
按下    防抖完成   触发   播放提示音  刷新完成  可接受
按钮     检测     单击      "滴"    墨水屏   下次输入
                                      显示
```

### 5.2 语音操作时序
```
0ms    800ms    1000ms   ...    松开    +200ms   +500ms   +1500ms   +3000ms
 │       │        │            │        │        │         │         │
 ▼       ▼        ▼            ▼        ▼        ▼         ▼         ▼
按下   确认    开始            结束    上传    识别     执行     更新
按钮   长按    录音            录音    完成    完成     完成     显示
       "叮~"   LED呼吸                           "叮"              新页面
```

### 5.3 同步时序
```
触发    +500ms   +1500ms   +2000ms   +3000ms
 │        │         │         │         │
 ▼        ▼         ▼         ▼         ▼
开始    检测      上传      下载      完成
同步    本地      变更      服务器    刷新
        变更                        显示
```
