## 2. 电源设计

针对本套由 ESP32-S3、4.2寸墨水屏及音频模组组成的低功耗物联网系统，电源架构设计如下：

### 2.1 总体电源拓扑 (Power Tree)

系统电源分为“常供电”与“受控供电”两个逻辑域，以实现深度睡眠时的零漏电。

```
[5V Type-C 输入]
      │
      ▼
[TP4056 充电管理] ──▶ [3.7V 锂电池 (自带保护板)]
      │
      ▼ 核心节点：VBAT (3.0V - 4.2V)
      │
      ├───▶ 【音频功放级】 ──▶ NS4168 (VCC直连VBAT)
      │                         └── (控制：GPIO 驱动 CTRL 引脚，平时拉低关断)
      │
      └───▶ 【系统主供电】 ──▶ RT9080-33GJ5 (LDO)
                                │
                                ▼ 节点：3.3V_UNMANAGED (待机功耗仅 2µA)
                                │
                                ├───▶ ESP32-S3 (主控 VDD)
                                │
                                └───▶ SGM2590D (负载开关) ◀── (控制：GPIO 驱动 EN)
                                          │
                                          ▼ 节点：3.3V_MANAGED (深睡彻底断电)
                                          ├───▶ INMP441 麦克风
                                          ├───▶ AHT21B 温湿度计
                                          └───▶ HINK 墨水屏 (逻辑与升压输入)
```

### 2.2 核心电路模块说明

#### 2.2.1 充电与电池管理 (TP4057)
* **功能**：负责将 USB 5V 转为锂电池 4.2V 充电电压。
* **特性**：带电池反接保护，SOT-23-6 封装节省空间。
* **注意**：2000mAh 电池建议通过 PROG 电阻设定 500mA 左右的充电电流。

#### 2.2.2 主稳压电源 (RT9080-33)
* **功能**：将波动的电池电压稳压至绝对稳定的 3.3V。
* **优势**：600mA 输出能力可抗住 ESP32-S3 开启 Wi-Fi 时的峰值电流；2µA 极低静态电流保证长效续航。
* **配置**：输入输出端各放置 1uF 贴片电容 。

#### 2.2.3 外设供电切断 (SGM2521)
* **功能**：作为“电源闸门”，在系统深睡时物理断开除 MCU 外的所有耗电器件。
* **优势**：自带软启动 (Soft-Start)，消除墨水屏上电瞬间的浪涌电流，防止主电压跌落导致 MCU 重启。
* **控制**：EN 引脚需接 100KΩ~1MΩ 下拉电阻，确保 MCU 睡着后外设默认断电。

#### 2.2.4 墨水屏升压子系统 (E-ink Boost) (这部分包含在屏幕驱动里)
* **架构**：利用驱动 IC 配合外部组件生成高压。
* **关键组件**：
    * **MOS管 (Q1)**: SI1308EDL (低内阻 N-MOS) 。
    * **电感 (L1)**: 47uH 一体成型电感。
    * **二极管 (D1-D3)**: MBR0530 肖特基二极管。
    * **滤波**: 升压输出端需使用 50V 耐压电容 (4.7uF/1uF)。

### 2.3 软件配合规范 (防漏电必备)

1.  **引脚隔离**：在关闭 SGM2521 进入 Deep Sleep 前，必须将所有连接外设的 GPIO（SPI、I2C）配置为“浮空输入”状态，防止电流通过数据线向外设倒灌（Ghost Powering）。
2.  **低压停机**：ADC 监测电池电压。当 VBAT < 3.3V 时，在墨水屏显示“请充电”图标后进入永久睡眠。
3.  **音频关断**：在不播放声音时，将 NS4168 的 CTRL 引脚置低，因其直连电池，不关断会产生显著漏电。
