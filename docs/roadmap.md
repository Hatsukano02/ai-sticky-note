## 开发依赖

### 硬件依赖
- 墨水屏驱动库：GxEPD2
- 音频库：ESP32 I2S驱动
- 网络库：ArduinoWebsockets

### 软件依赖
- ESP-IDF / Arduino框架
- Python 3.10+
- FastMCP
- SQLite / Redis

### 外部服务
- 豆包API（语音识别、对话）
- 千问API（备选）
- 天气API（OpenWeatherMap等）

## 风险预案

| 风险 | 应对策略 |
|------|----------|
| 墨水屏刷新慢 | 优化局部刷新，减少全屏刷新频率 |
| 语音识别不准 | 支持多模型切换，用户自定义关键词 |
| 续航不达预期 | 深度休眠策略，降低同步频率 |
| WiFi不稳定 | 断线重连机制，增强离线能力 |
