# AI冰箱贴

> 贴在冰箱上的智能生活助手

![AI冰箱贴概念图](docs/assets/concept.png)

## 它是什么？

AI冰箱贴是一款贴在冰箱门上的智能显示设备。它像一张电子便利贴，能显示日期、天气、待办事项、日程安排等信息。不同的是，你可以通过语音与它对话，让它帮你记录想法、设置提醒、切换显示内容。

不需要打开手机，不需要解锁屏幕。抬手一看，开口一说，生活信息尽在掌握。

## 核心特点

**📌 无感存在**

电子墨水屏常显内容，贴在冰箱上就像一张普通的便利贴。不会像手机屏幕那样打扰你，只在需要时为你呈现信息。

**🎤 语音交互**

长按按钮，直接说话：
- "提醒我下午3点开会"
- "距离考试还有多少天"
- "切换到天气"

不需要打字，不需要翻菜单。

**🔋 超长续航**

充一次电可以使用2-4周。它安静地待在那里，不需要你每天惦记充电。

**📱 多种信息显示**

- **日期页** - 大号日期显示，支持农历
- **天气页** - 当前温度、天气状况
- **待办页** - 待办事项清单
- **日历页** - 日程概览
- **倒计时页** - 重要日子倒数
- **节气页** - 中国传统节气
- **电量页** - 设备状态
- **同步页** - 连接状态

## 适合谁用？

- **忙碌的工作者** - 快速记录待办，不用掏手机
- **备考的学生** - 考试倒计时，时刻提醒目标
- **注重效率的人** - 语音交互，减少操作步骤
- **追求简约生活的人** - 低侵入感，不打扰的生活助手

## 使用场景

**早晨出门**
> 看一眼冰箱贴：今天15度，记得带外套。待办：下午3点开会。

**做饭时**
> 长按按钮："提醒我10分钟后关火"

**睡前**
> 快速扫一眼明天的日程，安心入睡

**设置倒计时**
> "设置考试倒计时30天" - 屏幕上显示大大的数字，每天提醒你

## 设计理念

我们相信，好的科技产品应该像空气一样——存在，但不打扰。

AI冰箱贴不追求功能繁多，而是专注于做一件事：在你需要时，用最简单的方式呈现最重要的信息。

没有复杂的菜单，没有频繁的推送，没有让人焦虑的红点。只有一个安静的屏幕，和一句简单的语音。
